# Instrucciones para Levantar el Proyecto

Este repositorio contiene un proyecto con un backend de Laravel y un frontend de Angular para gestionar vehículos, conductores y propietarios.

## Backend (Laravel)

### Requisitos

- [Composer](https://getcomposer.org/download/)

### Configuración de la Base de Datos

1. Abre el archivo `config/database.php` del proyecto Laravel.

2. En la sección de configuración para MySQL, define las credenciales de tu base de datos local. Utiliza el nombre `acme` para la base de datos.

### Pasos

3. Clona este repositorio en tu entorno local.

4. En la raíz del proyecto Laravel, ejecuta el siguiente comando para instalar las dependencias:

   ```bash
   composer install
   php artisan serve
